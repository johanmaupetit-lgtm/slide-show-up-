# slide-show-up-

Pour faire les slides de show up. Présentation Reveal.js déployée sur GitHub Pages.

## Accès en ligne

Une fois GitHub Pages activé (voir plus bas) :
**https://johanmaupetit-lgtm.github.io/slide-show-up-/**

## Édition

Tout le contenu est dans `index.html`. Chaque `<section>` = une slide.
Ouvre le fichier en local dans un navigateur pour voir le résultat sans rien installer.

## Raccourcis pendant la présentation

- `→` / `Espace` : slide suivante
- `←` : slide précédente
- `↓` / `↑` : slides verticales
- `F` : plein écran
- `S` : mode présentateur (notes)
- `Esc` : vue d'ensemble

## Activer GitHub Pages (une seule fois)

1. Repo → **Settings** → **Pages**
2. Source : **GitHub Actions**
3. Le workflow `.github/workflows/pages.yml` déploie automatiquement à chaque push sur `main`.
